  1. Project lambda
  
      Functional Programming with lambda Expression.
	  
  2. New Date/Time API

      Simpler Management of dates and times.

  3. Stream API

      Improved collection Mangement,parallel processing

  4. Nashorn Javascript Engine.
  
      Program in javascript , using java classes and objects.
	  
  5. Concurrency enhancements
